hi-nginx 使用说明

hi-nginx 使用说明

它是基于最新版NGINX的增强版，同时支持使用以下4种编程语言进行web应用开发：

  • C++
  • java
  • lua
  • python

从v-2.2开始，hi-nginx全新改版，支持各语言开发功能独立安装，暂停原有的会话和缓存功能，优化应用模块的加载方式，使之与模块开发语言生态环境结合更紧密。
